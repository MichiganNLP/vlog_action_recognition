# Identifying Visible Actions in Lifestyle Vlogs

This repository contains the dataset and code for our ACL 2019 paper:

[Identifying Visible Actions in Lifestyle Vlogs](https://arxiv.org/abs/1906.04236)

## Task Description

<img src="images/task_description.jpg" width="90%">

<p align="left"> Given a video and its transcript, which human actions are visible in the video? </p>
